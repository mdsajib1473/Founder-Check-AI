## 🚀 **How to Use (Quick Start)**

### **1. Start Servers**
```bash
# Terminal 1: Backend
cd m:\FounderCheck\backend
python main.py

# Terminal 2: Frontend
cd m:\FounderCheck\frontend
npm run dev
```

### **2. Create Analysis**
1. Open http://localhost:5173
2. Enter startup idea
3. Click "Analyze My Idea"
4. Wait for results (5-10 sec)

### **3. Export PDF**
1. Click "📥 Export Report" button
2. PDF downloads automatically
3. Open from Downloads folder

### **4. Share**
- Email to investors
- Present to mentors
- Keep in records
- Print if needed
